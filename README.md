## 破解

+ 将破解文件夹中的“license_standalone.lic”文件复制到软件安装目录中的 "licenses"文件夹内,默认安装路径为：C:\Program Files\MATLAB\R2018b\licenses

+ 将破解文件夹中的“netapi32.dll”文件复制粘贴到安装目录中的win64文件夹内即可，默认路径为C:\Program Files\MATLAB\R2018b\bin\win64
