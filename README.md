## 破解

1. 将破解文件夹中的“license_standalone.lic”文件复制到 软件安装目录 中的"\licenses\"文件夹内,
    + 一般路径为："安装路径\licenses"
    + 如果有其他lic文件请删除

2. 将破解文件夹中的“netapi32.dll”文件复制到 安装目录中 的"\bin\win64\"文件夹内即可，
    + 一般路径为："安装路径\bin\win64"
